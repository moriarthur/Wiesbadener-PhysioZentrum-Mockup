# Wiesbadener PhysioZentrum

Eine moderne Physiotherapie-Praxis in Wiesbaden.

## Projektübersicht

Dies ist eine vollständige, responsive Website für das Wiesbadener PhysioZentrum. Die Website wurde entwickelt, um Informationen über unsere Dienstleistungen, unser Team und Kontaktmöglichkeiten zu bieten.

## Funktionen

- **Responsive Design**: Optimiert für Desktop und Mobile
- **Moderne Technologien**: React, TypeScript, Tailwind CSS
- **Schnelle Ladezeiten**: Durch optimierten Build-Prozess
- **Barrierefreiheit**: Semantisches HTML und ARIA-Labels

## Installation

```bash
npm install
```

## Entwicklung

```bash
npm run dev
```

## Build für Produktion

```bash
npm run build
```

Die optimierten Dateien werden im `build`-Ordner abgelegt.

## Technologie-Stack

- **Frontend**: React 18 mit TypeScript
- **Styling**: Tailwind CSS mit benutzerdefinierten Komponenten
- **Build-Tool**: Vite 6
- **Icons**: Lucide React
